## Como Contratar um Agente

Contratar um agente no Ad Product Forge é o primeiro passo para automatizar tarefas no seu fluxo de trabalho. O processo é guiado por um assistente de contratação em 5 etapas.

### Pré-requisitos

- Acesso ao painel forge-admin
- Credenciais de provider de IA configuradas (OpenAI, Anthropic, etc.)
- Saldo disponível na empresa para o orçamento inicial do agente

### Passo 1: Informções Bsicas

1. Acesse a pgina de "Contratar Agente" no painel
2. Preencha o nome do agente (ex: "Code Review Bot", "Content Writer")
3. Adicione uma descrição clara do papel do agente
4. Selecione a provider de IA (OpenAI, Anthropic, etc.)
5. Escolha o modelo de IA (GPT-4, Claude-3, etc.)

### Passo 2: Configuração

1. Defina as instruções do sistema (o "prompts" base que orienta o comportamento do agente)
2. Configure o workspace onde o agente operar
3. Selecione as ferramentas iniciais que o agente poder usar
4. Defina horrios de disponibilidade (opcional)

### Passo 3: Contrato

1. Defina o orramento inicial do contrato
2. Configure o limite máximo de gasto por período
3. Escolha se o contrato deve ser renovado automaticamente
4. Revise os custos estimados com base no modelo escolhido

### Passo 4: Reviso

1. Revise todas as informações preenchidas
2. Verifique o orramento total
3. Confirme as ferramentas e permissões concedidas
4. Faça ajustes se necessário usando o boto "Voltar"

### Passo 5: Confirmao

1. Clique em "Confirmar Contratação"
2. Aguarde a inicialização do agente
3. Receba a confirmação de que o agente está ativo
4. O agente aparecer na lista de agentes ativos

### O que acontece depois?

- O agente fica disponível para receber tarefas
- Você pode monitorar o uso e orçamento em tempo real
- O contrato é renovado automaticamente se a opção estiver ativa
- Você pode ajustar o orçamento a qualquer momento

### Dicas Importantes

- **Comece com orelhas pequenas**: Defina orçamentos iniciais menores para testar o comportamento do agente
- **Revise as ferramentas**: Conceda apenas as ferramentas necessárias para cada agente
- **Monitore os primeiros dias**: Acompanhe de perto o uso nos primeiros dias para ajustar configurações
