### Glossário Rápido

| Termo          | Significado                             |
| -------------- | --------------------------------------- |
| **Agent**      | Agente de IA automatizado               |
| **Contract**   | Contrato que define orçamento e termos  |
| **Role**       | Função/papel com permissões específicas |
| **Tool**       | Ferramenta que o agente pode usar       |
| **Task**       | Tarefa criada para um agente executar   |
| **Top-up**     | Adicionar saldo ao contrato             |
| **Workflow**   | Fluxo de trabalho automatizado          |
| **Capability** | Capacidade/permissão para usar algo     |
