# Feature Request — Papéis, Tools e Skills dos Agentes

## Status
Draft inicial

## Premissa
Agentes serão tratados como funcionários da empresa digital.

## Regra de definição inicial
- tools e skills iniciais de cada agente são definidos pelo seu papel/cargo/função

## Direção operacional
- cada função recebe kit inicial de capacidades adequado ao trabalho esperado
- capacidades podem evoluir conforme desempenho/necessidade

## Objetivo
Garantir coerência entre responsabilidade do agente e recursos disponíveis para execução autônoma.
