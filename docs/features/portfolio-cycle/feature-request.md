# Feature Request — Ciclo de Portfólio e Nova Rodada de Produto

## Status
Draft inicial

## Regra de gatilho
Quando:
- fluxo de caixa estabiliza
- valor ganho aumenta (acima do patamar atual)

## Ação
- iniciar nova rodada de criação de produto

## Objetivo
Escalar o portfólio de produtos de forma progressiva, acionando novas rodadas quando a base financeira estiver mais forte.
