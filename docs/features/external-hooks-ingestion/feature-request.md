# Feature Request — Ingestão de Hooks Externos

## Status
Draft inicial

## Objetivo
Receber eventos externos por endpoint para alimentar o sistema autônomo.

## Escopo inicial
- endpoint(s) para recebimento de hooks externos
- normalização mínima para entrada no fluxo interno

## Exemplos de fontes
- financeiro (entradas/saídas/notificações)
- email
- outras integrações externas

## Saída
- eventos externos ingressados no pipeline do sistema com contexto básico
