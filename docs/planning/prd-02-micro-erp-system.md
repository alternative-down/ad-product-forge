## 1. Summary

### Classification: AD-PRODUCT-FORGE APPLICATION

**This PRD describes business operations infrastructure specific to ad-product-forge.** Financial tracking enables Nicolas to understand the economics of his autonomous agent platform (costs vs. revenue). This is application-specific, not framework infrastructure.

### Objective
Track simple financial metrics (expenses, revenues).

### Value (for ad-product-forge)
- Understand costs of running agents and services
- Track revenue generated by applications
- Simple forecasting for business planning

### Important Note
This is a personal project. Skip complex features like:
- Approval workflows
- Budget limits
- Compliance reporting
- Integration with external accounting software

## 2. Scope

### Included
- Log expenses (cost category, amount, description)

## 4. Database

```sql
CREATE TABLE financial_log (
  id TEXT PRIMARY KEY,
  type TEXT NOT NULL, -- 'expense' or 'revenue'
  amount DECIMAL(10, 2) NOT NULL,
  description TEXT,
  logged_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```
